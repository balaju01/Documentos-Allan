Supported Operating Systems
---------------------------
To be able to play Star Wars: Knights of the Old Republic, you
must have Windows 98, 98SE, Millennium Edition (ME), 2000, or
XP installed and configured correctly, and your computer hardware
must be OpenGL 1.4 and DirectX 9.0b compatible.

According to Microsoft, Windows 95 has entered its non-supported
phase as of December 2001. Because of this, Star Wars: Knights of
the Old Republic doesn't support this operating system. Windows 98
also entered its non-supported phase as of June 2003. Because of
this, several hardware manufacturers are no longer developing
drivers for this operating system.

Minimum System Requirements
---------------------------

Computer: 100% DirectX 9.0b compatible computer required.

CPU: Pentium III or Athlon class 1 GHz or faster CPU required.
Pentium 4 or Athlon XP class 1.6 GHz or faster CPU recommended.

Memory: 128 MB RAM required for Windows 98, 256 MB RAM required
for Windows ME/2000/XP. 512 MB RAM recommended.

Graphics Card: 32 MB OpenGL 1.4 compatible AGP or PCI 3D
Hardware Accelerator with Hardware Transform and Lighting (T&L)
Capability required.

Sound Card: 100% DirectX 9.0b compatible audio device required.

CD-ROM: Quad Speed IDE or SCSI CD-ROM drive required.

Input Device: Keyboard and mouse required.

DirectX: Microsoft DirectX 9.0b is included on this CD. You
will be prompted to install DirectX during installation of
this game.

NOTE: DirectX may require the "latest" updates for your
operating system. This may also include the latest drivers
for your particular motherboard, sound and video hardware.

Installation: Installation requires 4 GB of free hard drive
space. We recommend that you have at least an additional 200 MB
of free space available after installing the game for the
Windows swap/page file and save games.

General Information
-------------------

Installing Star Wars: Knights of the Old Republic
-------------------------------------------------
Insert the Star Wars: Knights of the Old Republic Install CD
into your CD-ROM drive. The launcher screen will open
automatically unless Autoplay is disabled. If it does not,
double-click on the My Computer icon, then the CD-ROM icon.
This should bring up the launcher menu. If it does not,
double-click on Autorun.exe in the window that opens. Click
on the top button on the launcher and the installer will
guide you through the installation process.

Installation prompts you for the installation location, asks
where to place shortcuts in the Start Menu and which profile
to allow access to this menu, which shortcuts to install, if
you want a shortcut on your desktop, and prompts you to install
DirectX 9.0b.

The default install path is C:\Program Files\LucasArts\SWKotOR.

Running Star Wars: Knights of the Old Republic
----------------------------------------------
To run the game, click on your Start Menu, select (All) Programs,
then Star Wars: Knights of the Old Republic, and click Star Wars:
Knights of the Old Republic. This will open the Launcher screen.

Desktop Color Palette
---------------------
We recommend that you set your desktop to True Color or Highest
(32 bit) color. When playing Star Wars: Knights of the Old
Republic with your desktop set to a color palette other than
32 bit, you may experience various problems while running the
game.
